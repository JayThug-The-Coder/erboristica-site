# Foto Open Graph (anteprime social)

Le foto in questa cartella sono usate dai social media e dalle app di messaggistica quando qualcuno condivide un link al sito Athena's.

## Specifiche

- **Dimensioni**: 1200 × 630 pixel (esatti)
- **Formato**: JPG (preferito) o PNG
- **Peso**: max 300 KB
- **Margini**: lascia 10% di "respiro" sui bordi (alcuni social ritagliano)
- **Testo**: minimo (su mobile le anteprime sono piccole)

## Come testare

Dopo che le foto sono online, prova un URL su:
- **Facebook**: https://developers.facebook.com/tools/debug/
- **LinkedIn**: https://www.linkedin.com/post-inspector/
- **Twitter**: https://cards-dev.twitter.com/validator

Su WhatsApp basta condividere il link in una chat: l'anteprima compare in pochi secondi.
